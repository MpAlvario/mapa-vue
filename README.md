# MapHub – Plataforma de Monitoreo Geoespacial

Sistema web desarrollado para la visualización, monitoreo y análisis de eventos georreferenciados en tiempo real mediante tecnologías modernas de cartografía digital.

## Características
- Visualización de incidentes en mapas interactivos.
- Monitoreo de unidades y patrullas en tiempo real.
- Mapas de calor (Heatmaps).
- Dashboard estadístico de incidentes.
- Gestión de zonas y cuadrículas tácticas.
- Visualización y seguimiento de rutas.
- Actualización automática de información.
- Predicción de zonas con mayor incidencia.
- Tecnologías Utilizadas

## 🎨 Diseño y Visualización
La interfaz fue diseñada considerando principios de usabilidad, accesibilidad visual y análisis geoespacial para facilitar la interpretación rápida de eventos en el mapa.

### Colorimetría

Se realizó un estudio de colorimetría para la selección de colores utilizados en la representación cartográfica, buscando:
- Mejorar la identificación visual de incidentes.
- Facilitar la diferenciación entre categorías.
- Mantener una adecuada legibilidad sobre distintas capas del mapa.
- Reducir la fatiga visual durante sesiones prolongadas de monitoreo.

### Clasificación de Incidentes
Los incidentes fueron organizados mediante un esquema de doble codificación visual:
- Codificación por categoría mediante iconografía específica.
- Codificación por color para indicar tipo o nivel de atención.

Este enfoque permite identificar rápidamente la naturaleza de un incidente sin necesidad de consultar información adicional.

## Módulos Principales
 ### Mapa Terrestre
Visualización de incidentes, patrullas y zonas operativas mediante cartografía interactiva.

### Mapa Marítimo
Monitoreo de operaciones y eventos en áreas marítimas.

### Análisis Geoespacial
Representación de información mediante capas temáticas, clusters y mapas de calor.

### Gestión Operativa
Seguimiento de personal, vehículos y recursos desplegados.
